== Changelog ==

= 2.0.0 =
* remake Divine with Kopa Framework 2.0.0
* upgrade plugin Divine Toolkit to 2.0.3
* attach plugin "Kopa Page Builder" - build your page quickly and smartly
* support import demo pages like as our demo pages
* remove layout "Contact Page", and add shortcode "Contact Form"
* fix some css bug, and custom color
* fix reponsive issue for page "Shop - Woocommerce"

* NOTE: Because Divine 2.0.0 is big update with new framework, please following guide:
- backup your site (theme & database) before update
- delete old version of plugin "Divine Toolkit"
- delete old version of theme "Divine"
- upload and isntall Divine 2.0.0
- go to "Appearance >> Install Plugins" to install Divine Toolkit & Kopa Page Builder
- rebuild your site with help from documents and video guide
